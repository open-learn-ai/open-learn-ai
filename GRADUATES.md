## Verification Process

### What We Check

1. **Authenticity**: Code is original, not copied from solutions
2. **Understanding**: Can explain their work (we may ask follow-up questions)
3. **Completion**: All required modules and capstone finished
4. **Quality**: Code is readable, documented, and works

### What We Don't Check

- Background or credentials
- Prior experience
- Formal education
- Where you're from

**The work speaks for itself.**

### Review Timeline

- Submit PR → Initial review within 1 week
- Questions/revisions → 1-2 weeks
- Approval → Listed immediately
